### Node

#### node 和 npm 在如何安装呢

1.首先安装 nvm ： node 的版本管理工具

```
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.2/install.sh | bash
```

2. 安装好 nvm 后，使用 nvm 安装 node 和 npm

nvm install v8.4.0

3. 这样既可安装 node 和相应的 npm 了


#### 删除 npm

rm -rf /usr/local/bin/npm

这样就可以把 npm 删除了
